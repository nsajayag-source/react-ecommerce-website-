# React E-Commerce Website

A responsive e-commerce web application built with React.js.

### Features
- Product listing page with images, price, and description
- **Search filter** - search products by name (e.g. "electronics" shows only electronics)
- **Category pages** - click category to view filtered products (Electronics page, etc.)
- Add to cart / Remove from cart functionality
- Cart page showing total price and quantity
- Navigation between pages using React Router
- Mobile responsive design

### Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- React Router
- React Hooks (useState, useEffect)

### How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
